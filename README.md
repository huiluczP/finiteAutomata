# finiteAutomata
Algorithms related to finiteAutomata,achieved by python.

DFA确定型有穷自动机和NFA非确定型有穷自动机相关算法实现。</br>
---
NFA相关功能：</br>
1. 利用五元组，创建NFA模型</br>
2. 输入字符串，判定是否被该NFA模型接受</br>

DFA相关功能：</br>
1. 利用五元组，创建DFA模型</br>
2. 输入字符串，判断是否被该DFA模型接受</br>
3. NFA转DFA</br>
4. DFA最小化</br>
5. RE与DFA转换</br>
---
test.py中有各功能使用示例，可参考。
